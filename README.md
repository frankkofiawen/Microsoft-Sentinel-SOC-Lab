## 📖 Overview

This project demonstrates the design and implementation of a Microsoft Sentinel Security Operations Center (SOC) lab.

The lab showcases how to:

- Deploy Microsoft Sentinel
- Configure Log Analytics Workspace
- Collect Windows Security Events
- Write Kusto Query Language (KQL) queries
- Create Analytics Rules
- Investigate Security Incidents
- Perform Threat Hunting
- Map detections to the MITRE ATT&CK Framework

The goal is to simulate the daily responsibilities of a Tier 1 / Tier 2 SOC Analyst using Microsoft Sentinel.
